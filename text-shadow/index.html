Text-shadow

text shadow is used to apply shadow in text. text-shadow makes word attractive to look .
The Syntax of text-shadow is almost similarly to Box Shadow, except Spread and inset as they are not allowed in Text Shadow. Text Shadow can enhace look of textcontent only, like p tag, headings etc.

Text Shadow Example

text-shadow: 2px 2px 2px red

text-shadow: 2px -2px 2px red

text-shadow: 10px 6px 6px rgb(49, 135, 233)

text-shadow: -2px -2px 2px blue;

Multiple Text Shadow

We can apply multiple text-shadow separated by ","

text-shadow: 1px 1px 2px red, 0 0 1em blue, 0 0 0.2em blue;
