ngmodel in angular

The ng-model directive binds the value of HTML controls (input, select, textarea) to application data.

ng-model Directive

With the ng-model directive you can bind the value of an input field to a variable created in Angular
ngmodel is used for two way data binding. 

<input type="text" class="form-control" id="name" required [(ngModel)]="data.name" /> 
  

<p>Hello {{ data.name }}!</p>
