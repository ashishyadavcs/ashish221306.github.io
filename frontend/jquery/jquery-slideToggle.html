jquery SlideToggle() | Jquery Slide Functions

Jquery Slide functions are used to show and hide an element with slide effect. This will decrease and increase height of element with transition. Fro smooth effect you can use slide Function with time parameter.

  1. Jquery SlideUp()
  2. Jquery SlideDown()
  3. Jquery SlideToggle()

Jquery SlideDown()

JQuery slideUp function is used to show or hide an html element with slide effect and transitions. The default duration is 400. But you can change duration according to your wants using "slow", "fast" or n milliseconds.

Syntax

$(slector).slideDown("slow");

for slow effect use slideToggle('slow')

Example

Lorem ipsum dolor sit amet consectetur adipisicing elit. Repudiandae accusamus perspiciatis sequi placeat eius architecto minima. Velit nisi, nobis, est maiores n ulla iure voluptatem dignissimos, dolores error atque temporibus omnis?

    
<button class="custom-button" id="sld">
slide Down
</button>
<div class="slide" id="sldd">
    <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Repudiandae accusamus perspiciatis sequi placeat eius architecto minima. Velit nisi, nobis, est maiores n
        ulla iure voluptatem dignissimos, dolores error atque temporibus omnis?</p>
</div>

    
<script>
    $('#sld').click(function(){
        $('#sldd').slideDown();
    })
</script>
